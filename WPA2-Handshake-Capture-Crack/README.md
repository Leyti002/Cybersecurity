# WPA2 Handshake Capture & Crack

**Course:** Cybersecurity  
**Student:** Oumar Leyti Ndiaye | A00228981

## Overview
Full demonstration of a WPA2 wireless network attack using
Airgeddon — from handshake capture to offline password cracking.

## Topics Covered
- 4-Way Handshake interception
- Deauthentication attack (aireplay-ng)
- Monitor mode activation (wlan0mon)
- Dictionary attack with Aircrack-ng (key found: success)
- Brute-force attack with Aircrack-ng + Crunch
- Wireshark capture validation
- Counter-measures: WPA3, PMF (802.11w), WIDS

## Tools Used
- Airgeddon
- Aircrack-ng suite (Airmon-ng, Airodump-ng, Aireplay-ng)
- Crunch
- Wireshark
- Kali Linux

## Key Results
- Handshake successfully captured from target network
- Password cracked in under 1 second via dictionary attack
- Brute-force demonstrated with Crunch (complexity analysis)
